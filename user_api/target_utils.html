class buildcc::TargetEnv¶

Public Functions

inline TargetEnv(const char *target_relative_to_env_root)¶

Change the relative root path for a particular Generator / Target.

Absolute root now changes to env::get_project_root_dir() / target_relative_to_env_root

Absolute build dir remains the same.

Can be used implicitly

Parameters

target_relative_to_env_root – Change root dir with respect to env::get_project_root_dir()
